# Eksperimen_SML_Muhammad_Ivan

Repository ini berisi hasil eksperimen preprocessing untuk mata kuliah Sistem Machine Learning.

## Link GitHub
- Repository: https://github.com/ivan8419/Eksperimen_SML_Muhammad_Ivan

## Struktur Folder
- `namadataset_raw/` - Dataset mentah
- `preprocessing/` - Script preprocessing
- `namadataset_preprocessing/` - Dataset hasil preprocessing
- `Eksperimen_Muhammad-Ivan.ipynb` - Notebook eksperimen
- `automate_Muhammad-Ivan.py` - Script otomasi
